# 第九章：程式設計思維(結合AI)

- 掌握問題分析與 I/O 定義的思維方式
- 學會內容生產的標準化與模板化
- 建立自動化流程設計與實現能力
- 理解 LLM 與程式的智能整合應用

## 實戰練習
1. 使用 I/O 分析框架解決實際問題

4. 結合 LLM 優化現有工作流程

- 如何選擇合適的問題進行優化？
- 什麼時候需要考慮標準化？
