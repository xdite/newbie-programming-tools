# 第一章：命令行基礎能力

## 學習目標
- 掌握基本的 shell 命令操作
- 理解並能夠管理環境變量
- 熟悉管道符和重定向的使用
- 能夠使用包管理器安裝和管理軟件

## 章節內容

### 1.1 Shell 基礎命令
- 什麼是終端機(Terminal)和 Shell
- 檔案系統導航 (cd, pwd)
- 檔案和目錄操作 (ls, mkdir, rm, cp, mv)
- 檔案內容查看 (cat, less, head, tail)
- 檔案權限管理 (chmod, chown)

### 1.2 環境變量
- 什麼是環境變量
- 查看環境變量 (env, echo)
- 設置環境變量 (export)
- PATH 變量的重要性
- 永久設置環境變量

### 1.3 管道符和重定向
- 管道符 (|) 的使用
- 輸入重定向 (<)
- 輸出重定向 (>, >>)
- 錯誤輸出重定向 (2>)
- 常見的管道命令組合

### 1.4 包管理器使用
- 包管理器的概念
- apt (Ubuntu/Debian) 的基本使用
- brew (macOS) 的基本使用
- 軟件安裝、更新和移除
- 倉庫源的管理
